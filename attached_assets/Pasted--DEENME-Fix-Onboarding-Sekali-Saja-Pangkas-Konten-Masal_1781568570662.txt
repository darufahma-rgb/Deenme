# DEENME — Fix Onboarding: Sekali Saja + Pangkas Konten

## Masalah
1. Onboarding muncul terus karena belum ada logic cek `hasOnboarded` di App.jsx
2. Konten 6 langkah terlalu banyak — dipangkas jadi 3 langkah saja

---

## Fix 1 — Update `src/App.jsx`

### A. Tambah import OnboardingOverlay (setelah import AdminPage):
```js
import { OnboardingOverlay } from './OnboardingPage.jsx';
```

### B. Tambah state showOnboarding (setelah state timezone):
```js
const [showOnboarding, setShowOnboarding] = useState(false);
```

### C. Di dalam useEffect load data dari Supabase/server, cek apakah user baru

Cari blok load data user (sekitar baris yang ada `user_data` atau `/api/user/data`).

Kalau pakai server API, cari:
```js
const res = await api('/api/user/data');
const json = await res.json();
```

Tambahkan logika cek onboarding SETELAH load data:
```js
const res = await api('/api/user/data');
const json = await res.json();
const d = json.data;

// Kalau tidak ada data ATAU belum pernah onboarding → tampilkan onboarding
if (!d || !d.hasOnboarded) {
  setShowOnboarding(true);
  // Kalau tidak ada data sama sekali, set defaults dulu
  if (!d) return;
}
```

Kalau masih pakai Supabase langsung, cari:
```js
const { data } = await supabase.from('user_data').select('data').eq('code_id', codeId).single();
```

Tambahkan setelahnya:
```js
if (!data?.data || !data.data.hasOnboarded) {
  setShowOnboarding(true);
  if (!data?.data) return;
}
```

### D. Tambah handler onOnboardingDone (setelah onLogout function):
```js
const onOnboardingDone = async (selectedTz) => {
  setTimezone(selectedTz);
  setShowOnboarding(false);
  // Simpan flag hasOnboarded bersama data yang ada
  try {
    await api('/api/user/data', {
      method: 'POST',
      body: { payload: { hasOnboarded: true, lastSaved: new Date().toISOString() } },
    });
  } catch {}
};
```

Kalau masih pakai Supabase langsung:
```js
const onOnboardingDone = async (selectedTz) => {
  setTimezone(selectedTz);
  setShowOnboarding(false);
  await supabase.from('user_data').upsert(
    { code_id: codeId, data: { hasOnboarded: true } },
    { onConflict: 'code_id' }
  );
};
```

### E. Render OnboardingOverlay di LUAR deenme-root

Cari:
```jsx
return (
  <div className="deenme-root">
```

Ganti menjadi:
```jsx
return (
  <>
    <div className="deenme-root">
      {/* ... semua konten yang sudah ada ... */}
    </div>

    {showOnboarding && (
      <OnboardingOverlay
        userName={userName}
        onDone={onOnboardingDone}
      />
    )}
  </>
);
```

**PENTING:** Tutup `<div className="deenme-root">` dengan `</div>` sebelum `{showOnboarding && ...}`, lalu tutup keseluruhan dengan `</>`
